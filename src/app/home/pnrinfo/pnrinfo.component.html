What is PNR? PNR stands for Passenger Name Record, is a unique 10 digit number which contains reservation details of passengers in Indian Railway. When a passenger buy any reservation ticket in Indian Railway, all details are stored in a centrailized database. These detail includes:
  • Passenger Details - Name, Age, Sex, ID details
  • Train Details - Train Number, Train Name, Booking Status
  • Payment Details - Ticket fare, Transaction details
